Exploratory Data Analysis (EDA):
Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on an e-commerce sales dataset using Microsoft Excel. The objective of the analysis was to identify customer purchasing patterns, product demand trends, payment preferences, order status distribution, and high-value transactions.

The project demonstrates the practical application of descriptive statistics and data analysis techniques to extract meaningful business insights from raw data.

Objectives:
Perform data exploration and analysis
Calculate descriptive statistics
Identify trends and patterns in customer behavior
Detect outliers in transaction data
Generate meaningful business insights

Dataset Details:

The dataset contains transactional information related to:

Customer orders
Product purchases
Quantity and pricing
Payment methods
Order status
Referral sources
Total transaction values
Total Records
1200 rows of transactional data

Tools & Techniques Used:
Microsoft Excel
Excel Formulas
Data Filtering
Conditional Formatting
Descriptive Statistics

Trend Analysis:

The analysis identified several important trends:

Electronic products such as printers, monitors, phones, and tablets are frequently purchased.
Digital payment methods are widely preferred by customers.
Most orders were successfully delivered.

Outlier Detection:

Outlier analysis was performed on the TotalPrice column using Conditional Formatting in Excel.

Method Used:
Conditional Formatting → Highlight Cells Rules → Greater Than → 4000

This helped identify unusually high-value transactions in the dataset.

Conclusion:
This project used Exploratory Data Analysis (EDA) in Microsoft Excel to identify patterns, trends, and customer behavior from the dataset.
