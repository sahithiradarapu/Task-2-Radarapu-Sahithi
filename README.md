# 🚀 Data Analytics Project 2 – Exploratory Data Analysis (EDA)

This project was completed as part of my Data Analytics Internship at DecodeLabs.

The objective of this project was to perform Exploratory Data Analysis (EDA) on an e-commerce sales dataset using Microsoft Excel. The analysis focused on identifying customer purchasing behavior, sales trends, payment preferences, and high-value transactions through descriptive statistical techniques.


# 🔍 Key Tasks Performed

✅ Calculated statistical measures such as:
- Mean
- Median
- Count
- Minimum & Maximum values

✅ Identified:
- Frequently purchased products
- Customer purchasing trends
- Payment method preferences
- Order delivery trends

✅ Performed Outlier Detection using:
- Conditional Formatting in Microsoft Excel

✅ Generated meaningful business insights from raw transactional data


# 🛠 Tools Used

💻 Microsoft Excel  
📊 Excel Formulas  
📈 Filters & Conditional Formatting  


# 📈 Key Insights

🔹 Electronic products were purchased more frequently  
🔹 Digital payment methods were widely preferred  
🔹 Most orders were successfully delivered  
🔹 High-value transactions were identified as outliers  


# 🎯 Project Outcome

Successfully analyzed and interpreted raw business data to uncover trends, patterns, and customer behavior insights that support data-driven decision-making.

This project strengthened my understanding of:
📊 Exploratory Data Analysis (EDA)  
📊 Descriptive Statistics  
📊 Data Interpretation  
📊 Analytical Thinking  
📊 Business Insight Generation  

✨ A valuable hands-on learning experience that enhanced my practical data analytics skills.
